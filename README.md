# 👤 User Management Platform

A full-stack User Management System built with **Spring Boot**, **React**, and **PostgreSQL**.  
The project demonstrates a clean REST API architecture with layered backend design and modern frontend integration.

---

## 🚀 Features

- Create users
- View all users
- Update user information
- Delete users
- Form validation (frontend + backend)
- REST API communication
- Cross-Origin (CORS) enabled
- DTO-based architecture

---

## 🧰 Tech Stack

### Backend
- Java 17+
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven

### Frontend
- React
- Axios
- JavaScript (ES6)
- HTML / CSS
